Directorio de Recursos

Desarrollo Web:

Tutoriales:

  • MDN Web Docs - Recursos exhaustivos sobre tecnologías web, desde HTML hasta JavaScript.
  • freeCodeCamp - Plataforma interactiva con tutoriales sobre desarrollo web y programación.

Herramientas:

  • CodePen - Editor de código en línea para prototipado rápido de HTML, CSS y JavaScript.
  • Can I Use - Herramienta para verificar la compatibilidad de características web en diferentes navegadores.

Articulos:

  • Smashing Magazine - Revista en línea con artículos sobre diseño web, desarrollo front-end y UX/UI.

Ciencia de Datos y Machine Learning:

Tutoriales:

  • Kaggle - Plataforma con tutoriales y competiciones de ciencia de datos y machine learning.
  • DataCamp - Cursos interactivos sobre ciencia de datos, estadísticas y machine learning.

Herramientas:

  • Jupyter Notebook - Entorno interactivo para escribir y ejecutar código en Python, ideal para análisis de datos.
  • TensorFlow - Biblioteca de código abierto para machine learning desarrollada por Google.

Articulos:

  • Towards Data Science - Plataforma de Medium con artículos sobre ciencia de datos, machine learning y análisis de datos.
